What is MoodSwing?

MoodSwing is a PC game to be built upon Microsoft's XNA Framework (version 3.1). It is currently planned to be an RTS-simulation game hybrid in full 3D. The project has a git repository hosted by Github here.

The Development Team

MoodSwing's development team consists of four students — amateur programmers and artists — whose names appear in the footer.

What Is This Site For?

This site is used as a communication channel and a development tracker for the designers, architects, developers, and spectators of the MoodSwing project.
